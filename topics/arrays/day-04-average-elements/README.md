## Problem
Average of all the elements in the array .

## Approach
Used for loop traverse through the array and while traversing maintained a variable for storing sum of the elements of the array
After completing the traversal simply divide the sum by no. of elements in the array.
## Complexity
Time Complexity: O(n) As we are traversing the array once.
Space Complexity: O(1). No addition space being used.
